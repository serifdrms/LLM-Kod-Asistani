# Gemini API Tabanlı Türkçe Kod Asistanı

Bu proje, **Mühendislikte Bilgisayar Uygulamaları** dersi için kapsamında geliştirilmiştir.

## Proje Hakkında
Kullanıcıların Türkçe ifadelerle (örn: "Yılan oyunu yap"), Python kodu üretmelerini sağlayan ve karmaşık kodları açıklayan yapay zeka destekli bir web uygulamasıdır. Google Gemini 2.5 Flash modeli kullanılmıştır.

##  Kurulum

1. Gerekli kütüphaneleri yükleyin:
   ```bash
   pip install -r requirements.txt

2. API Anahtarınızı app.py dosyasına girin.

3. Çalıştırın:
python -m streamlit run app.py
